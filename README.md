# Hamza Syed Premium Portfolio Website

A static multi-page portfolio website for GitHub Pages.

## Pages
- Home
- About
- Experience
- Projects
- Gallery
- Contact

## Deploy on GitHub Pages
1. Create a public GitHub repository, for example `hamza-portfolio`.
2. Upload all files from this folder.
3. Go to Settings > Pages.
4. Select Branch: `main` and Folder: `/root`.
5. Open your GitHub Pages URL.

## Add your images
Put images in the `assets` folder and update CSS backgrounds or HTML image tags.
